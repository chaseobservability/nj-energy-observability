# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

We expect all participants to be respectful, constructive, and professional.

Unacceptable behavior includes harassment, discrimination, and abusive conduct.

Maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that are not aligned with this Code of Conduct.

If you have concerns, please open an issue or contact the maintainer privately.
